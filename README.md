# Updating the Favicon

Replace the favicon files inside:

```bash
/public
```

Usually:

```bash
/public/favicon.ico
```

or

```bash
/src/assets/favicon.png
```

Then update the reference inside:

```html
index.html
```

Example:

```html
<link rel="icon" type="image/png" href="/favicon.png" />
```
